# atuin

Unofficial Chart for Atuin, the magical shell history.
The server provides fully encrypted synchronization of the shell history across machines.
https://github.com/ellie/atuin

[![Latest version](https://img.shields.io/badge/latest_version-0.1.0-blue)](https://artifacthub.io/packages/helm/rm3l/atuin)

## Installation

```bash
$ helm repo add rm3l https://helm-charts.rm3l.org
$ helm install my-atuin rm3l/atuin --version 0.1.0
```
